# Ejercicio de Colaboración con GitHub Codespaces

Repositorio: [colaboracion-codespaces](https://github.com/IvanJGG/colaboracion-codespaces.git)

---

## Instrucciones

1. Hacer un **fork** del repositorio:  
   [https://github.com/IvanJGG/colaboracion-codespaces.git](https://github.com/IvanJGG/colaboracion-codespaces.git)

2. Abrir el fork en **Codespaces**.  

3. Crear un archivo llamado **saludo.py** que:  
   - Pida al usuario su nombre.  
   - Muestre un mensaje de bienvenida.  
   - Ejemplo de salida:  
     ```
     Ingresa tu nombre: Ivan
     ¡Hola Ivan, bienvenido al repositorio colaborativo!
     ```
